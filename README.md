# Interactive Todo Card

This folder contains an interactive todo card component built with HTML, CSS, and vanilla JavaScript.

## Features

- **Editable Content**: Click the "Edit" button to modify the task's title, description, priority, and due date.
- **Status Transitions**: Change the task's status between "Pending", "In Progress", and "Done".
- **Priority Indicator**: A visual indicator shows the task's priority level (Low, Medium, or High).
- **Expand/Collapse Description**: Long descriptions can be expanded and collapsed.
- **Dynamic Time Management**: The time remaining until the due date updates automatically, and an "Overdue" indicator is displayed when the due date has passed.
- **Responsive Design**: The card is designed to work well on mobile, tablet, and desktop screens.
- **Accessibility**: The component includes ARIA attributes and is keyboard-navigable.
